Principais formataçoes

negrito / destaque

Nesta frase temos uma palavra em negrito, usando a tag b.(nao semantica)

Nesta frase tenho um texto em destaque usando a tag strong.(semantica)

Italico / ênfase

Nesta frase temos uma palavra em italico, usando a tag i.(semãntica)

Nesta frase temos um texto em ênfase usando a tag em.(semantica)

Texto marcado

podemos tambem criar um texto marcado usando a tag mark.

Texto pequeno

Estamos criando um texto pequeno neste paragrafo, usando a tag small.

Texto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado.

texto inserido

Podemos marcar um texto inserido para dar uma enfase e indicar que ele foi adicionado depois.

Existe tambem o sublinhado com a tag u.

Texto subrescrito

Para inserir coisas do tipo x20+3, usar a tag sup.

Texto subscrito
